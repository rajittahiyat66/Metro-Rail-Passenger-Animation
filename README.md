# 🚆 Metro Rail Passenger Full Animation Using OpenGL

A simple and interactive 2D Metro Rail Passenger Animation built using Python and OpenGL (PyOpenGL). This project demonstrates basic computer graphics concepts such as animation, passenger movement, train simulation, object rendering, and real-time graphics processing.

---

# 📌 Project Overview

The Metro Rail Passenger Full Animation project is a real-time computer graphics simulation where a metro train arrives at a station, stops automatically, opens its doors, allows passengers to exit and enter, then continues moving again.

The project is developed using Python OpenGL (PyOpenGL) and demonstrates smooth animation techniques and graphics rendering in a metro station environment.

This simulation includes:

- Moving metro rail animation
- Passenger entry & exit animation
- Automatic door opening and closing
- Metro station environment
- Smooth rendering using OpenGL
- Real-time animation system

---

# ✨ Features

## 🚆 Metro Rail Movement

The metro train continuously moves from left to right.

When the train reaches the station:

- The train automatically stops
- Doors open automatically
- Passenger exchange happens
- Doors close automatically
- Train starts moving again

---

## 🚪 Automatic Door Animation

The metro doors:

- Open smoothly
- Stay open during passenger movement
- Close smoothly before departure

---

## 🧍 Passenger Exit Animation

An inside passenger:

- Walks toward the door
- Exits the train
- Disappears after leaving

---

## 🧍‍♀️ Passenger Entry Animation

An outside passenger:

- Walks toward the train
- Enters inside the metro
- Disappears after entering

---

## 🚉 Metro Station Environment

The simulation includes:

- Station Platform
- Roof Structure
- Pillars
- Railway Track
- Sky Background
- Ground Area

---

## 🎨 Smooth Rendering

Uses:
